### About
- Small game based on dodging enemies and collecting energy

### Try it online [here](http://code.mar21.eu/JSRAIN/)
### Freatures
- Parallax background
- Ingame notifications
- Persistent storage
- Gesture/Button control
- Scene/Overlay based rendering

### Requirements
- Android 6.0 / SDK v23
- Java 1.8
