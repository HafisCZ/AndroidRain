### About
- Java (Android) game created using `OpenGL ES`
- Port of Java Rain
- ##### Planned features:
  - 2D sprite based
  - Singleplayer
  - Gesture/Motion control
  - Persistent storage
  - Improved game system
- ##### Not guaranteed features:
  - Ghost multiplayer
